# 🤖 AI Code Error Detector

An AI-powered web application that analyzes source code written in multiple programming languages and provides intelligent feedback on syntax errors, logical issues, code quality, and best practices.

## 🚀 Features

- 🌍 Supports multiple programming languages
  - Python
  - Java
  - C
  - C++
  - JavaScript
  - And more...
- 🔍 Detects syntax and logical errors
- 💡 Provides explanations in simple English
- ✅ Suggests corrected code
- 📊 Highlights code quality improvements
- 🎨 Clean and responsive user interface

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/ai-code-error-detector.git
```

### Navigate to the project

```bash
cd ai-code-error-detector
```

### Install dependencies

```bash
cd backend

pip install -r requirements.txt
```

---

## 🔑 Configure API Key

Create a `.env` file inside the `backend` folder.

```env
OPENAI_API_KEY=your_api_key_here
```

---

## ▶️ Run the Backend

```bash
uvicorn main:app --reload
```

Backend URL

```
http://127.0.0.1:8000
```

API Documentation

```
http://127.0.0.1:8000/docs
```

---

## 💻 How to Use

1. Open the frontend.
2. Select the programming language.
3. Paste your source code.
4. Click **Analyze**.
5. View the AI-generated report.

The AI provides:

- Syntax Errors
- Logical Errors
- Runtime Issues
- Best Practices
- Code Smells
- Optimized Code Suggestions
